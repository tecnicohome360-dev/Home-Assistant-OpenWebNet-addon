# Home360 OpenWebNet Add-ons for Home Assistant

Custom add-on repository for Home Assistant. Add this repository to the Add-on
Store to install the add-ons below.

## Add-ons

| Add-on | Description |
|--------|-------------|
| [Open Web Net USB Gateway](./openwebnet) | Turn a BTicino/Legrand L4686SDK USB adapter into an OpenWebNet TCP gateway (socat → TCP port 20000). |

## How to add this repository to Home Assistant

1. In Home Assistant go to **Settings → Add-ons → Add-on Store**.
2. Click the **⋮** menu (top right) → **Repositories**.
3. Paste the URL of this repository:

   ```
   https://github.com/tecnicohome360-dev/Home-Assistant-OpenWebNet-addon
   ```

4. Click **Add**, then close the dialog.
5. The **Open Web Net USB Gateway** add-on now appears in the store. Open it and
   click **Install**.

> Requires Home Assistant with the **Supervisor** (Home Assistant OS or Supervised).
